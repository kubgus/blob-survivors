# Blob Survivors

Simple evolution game about blobs made in Godot.

## Controls 🎮

- <kbd>W</kbd><kbd>A</kbd><kbd>S</kbd><kbd>D</kbd> - Move camera
- <kbd>Q</kbd><kbd>E</kbd> - Lift camera
- <kbd>⬆</kbd> - Fast forward time
- <kbd>⬇</kbd> - Slow down time
- <kbd>Esc</kbd> - Quit
